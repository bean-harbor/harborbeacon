# HarborDesk Angular Workspace

This workspace contains the formal HarborDesk admin web UI skeleton.

## Scripts

- `npm install`
- `npm start`
- `npm run build`

## Development Proxy

The dev server proxies same-origin admin API calls from `/api` to `http://127.0.0.1:8080`.

## Output

Production builds are written to `dist/harbordesk`.
