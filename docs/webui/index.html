Harbor Assistant Overview

One release shell for HarborGate, HarborBeacon, HarborOS, and AIoT.

Harbor Assistant is the unified admin surface for binding, account governance, task approvals, device operations, policy defaults, and system settings. The product is intentionally release-oriented: the UI exposes real state, surfaces backend gaps, and keeps day-to-day operator work in one place.
